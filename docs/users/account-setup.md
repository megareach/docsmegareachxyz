# Account Setup Guide

Complete guide to setting up your MegaReach account for maximum growth potential.

## Prerequisites

Before starting, ensure you have:
- Active Twitter/X account
- Email address for registration
- Wallet for blockchain features (optional)
- 10 minutes for initial setup

## Step 1: Registration

### Creating Your Account

1. **Visit MegaReach**
   - Go to [app.megareach.xyz](https://app.megareach.xyz)
   - Click "Get Started" button

2. **Choose Sign-Up Method**
   - Email & Password
   - Google OAuth
   - Twitter/X OAuth
   - Wallet Connect (Web3)

3. **Complete Registration**
   ```
   Required Information:
   - Username (unique)
   - Email address
   - Password (min 8 characters)
   - Agree to Terms of Service
   ```

4. **Verify Email**
   - Check inbox for verification email
   - Click verification link
   - Return to dashboard

## Step 3: Social Account Connection

### Connecting Twitter/X

1. **Navigate to Connections**
   - Settings → Connected Accounts
   - Click "Connect Twitter/X"

2. **Authorize MegaReach**
   - Redirected to Twitter
   - Review permissions
   - Click "Authorize app"

3. **Configure Permissions**
   ```
   Permissions Granted:
   ✓ Read tweets and profiles
   ✓ Post tweets and replies
   ✓ Access follower lists
   ✓ Manage relationships
   ✓ Access analytics
   ```

4. **Set Primary Account**
   - Select main account
   - Configure posting defaults
   - Set engagement limits

### Multiple Account Management

Add additional accounts:

1. **Add Account**
   - Click "Add Another Account"
   - Repeat authorization process
   - Assign account role

2. **Account Roles**
   ```
   Personal: Individual growth
   Business: Brand account
   Support: Customer service
   Content: Publishing only
   ```

3. **Account Settings**
   - Individual limits
   - Specific workflows
   - Separate analytics
   - Team access

## Step 5: Security Setup

### Two-Factor Authentication

Enable 2FA for account security:

1. **Navigate to Security**
   - Settings → Security
   - Click "Enable 2FA"

2. **Choose Method**
   - Authenticator app (recommended)
   - SMS verification
   - Email codes

3. **Setup Process**
   ```
   Authenticator Setup:
   1. Install Google Authenticator/Authy
   2. Scan QR code
   3. Enter verification code
   4. Save backup codes
   ```

### API Key Management

For developer access:

1. **Generate API Key**
   - Settings → API
   - Click "Create New Key"
   - Name your key
   - Set permissions

2. **Key Security**
   ```
   Best Practices:
   - Never share keys
   - Use environment variables
   - Rotate regularly
   - Monitor usage
   ```
